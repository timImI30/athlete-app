# athlete-app
Diese App begleitet Jugend-Athleten im Unihockey. Per täglichem Check-in erfassen sie Schlaf, Stimmung und Intensität. Ein Dashboard zeigt Trends der letzten Tage, während ein automatischer Warnmodus rechtzeitig vor Überlastung schützt. Athleten haken eigene Ziele ab und der Coach steuert das Training über eine direkte Feedback-Sektion.
